# Dashboard Tabs Sample

## What This Sample Demonstrates

`useDashboardTabs.example.tsx` shows a public-safe React hook for orchestrating portfolio dashboard tabs. It demonstrates tab metadata, lazy loading signals, privacy-aware tab visibility, and refresh callbacks.

It is an authored example and is not copied from production source.

## What Has Been Sanitized

- Tab IDs are generic.
- Data loading callbacks are placeholders.
- Wallet and bundle IDs are fake.
- No real Redux selectors, route names, or API endpoints are included.

## What Private Data Must Never Be Included

- Production route inventories.
- Private component names if they expose confidential implementation structure.
- Real wallet IDs.
- Real customer balances.
- Real transaction data.
- Real API responses.
- Backend URLs or tokens.
