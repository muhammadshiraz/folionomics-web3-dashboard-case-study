# Chart Data Transform Sample

## What This Sample Demonstrates

`chartDataTransform.example.ts` shows public-safe utilities for transforming fake portfolio rows into treemap data and historical chart data.

It is an authored example and is not copied from production source.

## What Has Been Sanitized

- Asset rows use fake sample values.
- Wallet labels and chain names are generic.
- No provider responses or production graph contracts are included.
- No real balances, prices, addresses, or transaction data are included.

## What Private Data Must Never Be Included

- Real wallet holdings.
- Raw API responses.
- Customer balances.
- Production provider schemas.
- Transaction hashes.
- Backend URLs or credentials.
