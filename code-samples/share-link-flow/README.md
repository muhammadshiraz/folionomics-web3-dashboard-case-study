# Share Link Flow Sample

## What This Sample Demonstrates

`shareLinkFlow.example.tsx` shows a public-safe React example for configuring share-link privacy settings, preparing a sanitized payload, and adapting fake portfolio data for a public view.

It is an authored example and is not copied from production source.

## What Has Been Sanitized

- The public URL uses `https://folionomics.example`.
- Wallet IDs, bundle IDs, balances, and assets are fake.
- No QR code, share token, backend URL, or API response is included.
- Privacy controls are reduced to simple booleans for demonstration.

## What Private Data Must Never Be Included

- Real share links.
- Real QR codes.
- Real wallet addresses.
- Real balances.
- Transaction hashes.
- Customer profile data.
- Backend URLs.
- Bearer or refresh tokens.
