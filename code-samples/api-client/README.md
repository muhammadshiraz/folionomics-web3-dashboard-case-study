# API Client Sample

## What This Sample Demonstrates

`apiClient.example.ts` shows a public-safe TypeScript API client wrapper for JSON requests, query parameters, auth token attachment, abort signals, defensive response parsing, and normalized HTTP errors.

It is an authored example and is not copied from production source.

## What Has Been Sanitized

- Backend URL is `https://api.example.invalid`.
- Auth token is passed as a placeholder function argument.
- Endpoint paths are generic examples.
- Response data is fake and minimal.
- No private infrastructure or production API contracts are included.

## What Private Data Must Never Be Included

- Real backend URLs.
- API keys.
- RPC keys.
- Bearer tokens.
- Refresh tokens.
- Production endpoint inventories.
- Raw API responses.
- Customer wallet data.
- Database URLs.
- Private infrastructure URLs.
